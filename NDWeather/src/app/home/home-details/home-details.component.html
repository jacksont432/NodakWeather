Nodak Weather A Weather Application Built For The ND Resident

Welcome to Nodak Weather! This website holds archived weather data for the state of North Dakota. It is in Version 2.0 with the ability to view data in a graph or table. All of the weather can be selected by region and type. In future releases we will add the ability to login to keep weather data as well as an option to become verrified and upload your own weather data.
